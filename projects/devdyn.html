DevDynasty: Full-Stack Ascent

Three.js / JavaScript / HTML / CSS / Node.js / Express / Blender - Animation, Set Building, Lighting

DevDynasty is a web game where users play as a CS major undergoing the application process for a full-stack developer internship. Like in the real world, the player must pass a resume screening and interview to receive an offer. This game features 3D scenes, animations, interactive pages with music, and interview questions pulled from a database.

I collaborated in a team of five to design and develop DevDynasty. Driven by my determination to elevate our original idea into an immersive 3D experience, I took on the challenge of learning and implementing Three.js during the month-long development of this project. This game was inducted into Oregon State's CS 290 - Web Development Hall of Fame in December 2023.
