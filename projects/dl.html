Applied Deep Learning

Python / PyTorch / Weights & Biases

This coursework is ongoing. Here are my projects from CS 435 - Applied Deep Learning with Stefan Lee.

Building a Scalar Autograd Framework

Learned about the techniques for autograd software by implementing a minimal automatic differentiation engine. This included dynamically constructing a computational graph, implementing forward and backward passes, and performing backpropagation. I also added gradient descent using the audograd system and wrote unit tests to validate gradient correctness.

Fully-Connected Network in PyTorch for 1D Functions

Implemented a fully-connected neural network in PyTorch to model 1D functions. This involved defining custom datasets and a feed-forward network class, implementing the training loop, addon experimenting with network depth, activation functions, learning rate schedules, and hyperparameter tuning.

Basic Convolutional Neural Networks and Logging

Built and trained a basic CNN with data augmentation and accuracy evaluation. Integrated Weights & Biases for experiment tracking and comparison across runs, enabling systematic analysis of model performance and training behavior.
