Funny How Time Slips Away (WIP)

Character Modeling / Rigging / Animation / Camera Layout / Set Building / Lighting / Particle Systems / Editing

A short film about the drawbacks of being a loving machine.

I have started working on a short that uses the same robot dog from my first 3D animation, but introduces more complicated scenes and characters with particles. It goes beyond the boundaries I set for my first animation, complete with an original story. So far, I've learned how to use Blender's particles system, rigify, and cloth system.

External materials used:
- Retriever puppy
- Sliding door
- Background chair
- Background table
- Background plant
- Curtain flannel
